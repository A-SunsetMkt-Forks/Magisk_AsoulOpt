# AsoulOpt
Android游戏线程调整程序，支持主流游戏，当然也支持一些小众游戏

## 主要功能
- 获取顶层应用包名，实现不同游戏不同方案
- 游戏重要线程专属核心，不被其他线程抢占
- 自适应放置方案，可判断未命名重负载线程
- 可不搭配第三方调度使用，但建议搭配调度以解决某些系统的负优化，通常不会与第三方调度冲突
